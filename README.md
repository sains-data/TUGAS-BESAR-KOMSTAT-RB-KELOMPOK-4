Repositori ini berisi pengumpulan tugas besar mata kuliah **Komputasi Statistika** dengan fokus pada penerapan **teknik data wrangling**.  
Dataset yang digunakan berasal dari **Badan Pusat Statistik (BPS)** mengenai produksi tanaman sayuran di Provinsi Sumatera Selatan tahun 2023.  

Proses wrangling meliputi:
- Pembersihan baris non-data  
- Konversi tipe data  
- Penanganan missing values  
- Pembentukan variabel turunan (total produksi, total cabai, persentase kontribusi cabai, total jamur)  
- Reshaping data (wide â†’ long)  
- Visualisasi komparatif antar kabupaten/kota
